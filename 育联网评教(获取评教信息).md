## 接口名 ##
    获取评教信息
### 说明 ###
    通过获取评教信息接口获取到所有的评教信息
### 接口地址 ###
```
POST http://educiot.com:32070/wxw/eva/tostudentevaluate?ids=-8864473216238468886%2C-8531883763910424235%2C-8075828664692882463%2C-4750163117504313856%2C-2936105965954905028%2C1564578188391632619%2C3053714649022220133%2C5043296452275618906%2C5905783283421694226&tid=511170835252661497
```
`ids通过获取评教成员信息接口获取(rid),每个rid之间需使用 "%2C" 拼接,最后一个tid通过获取评教列表获取(relationid)`

```
Accept-Encoding :gzip, deflate
Connection :close
Content-Type :application/x-www-form-urlencoded
Accept :*/*
Host :educiot.com:32070
User-Agent :yu lian wang/2.3.4 (iPhone; iOS 13.5; Scale/2.00)
Content-Length :14
Accept-Language :zh-Hans-HK;q=1, en-HK;q=0.9
FDtoken :4b17dff5cdf74076994768f33414673d
```
`FDtoken需要通过登录(login)接口获取`
